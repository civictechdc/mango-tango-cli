# CIBMangoTree N-grams Analyzer

N-gram analysis with multilingual tokenization support.

## Description

The n-gram analysis extracts n-grams (sequences of n words) from the text data and counts the occurrences of each n-gram in each message, linking the message author to the ngram frequency.

## Installation

```bash
pip install cibmangotree-analyzer-ngrams
```

## Features

- Unicode-aware tokenization
- Multilingual support (Latin, CJK, Arabic scripts)
- Social media entity preservation (hashtags, mentions, URLs)
- Configurable n-gram length (min and max)
- Statistical analysis of n-gram distributions

## Usage

This analyzer is automatically discovered by CIBMangoTree when installed.

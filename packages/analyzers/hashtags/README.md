# CIBMangoTree Hashtags Analyzer

Hashtag analysis for measuring coordination via hashtag usage concentration.

## Description

Analysis of hashtags measures the extent of online coordination among social media users by looking at how the usage of hashtags in messages changes over time. It measures whether certain hashtags are being used more frequently than others (i.e., trending).

## Installation

```bash
pip install cibmangotree-analyzer-hashtags
```

## Features

- Gini coefficient calculation for hashtag distribution
- Time-windowed analysis of hashtag concentration
- Interactive visualizations showing coordination patterns

## Usage

This analyzer is automatically discovered by CIBMangoTree when installed.
